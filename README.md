## wut this
a remake of flappy bird with different assets and some personalization
a super duper cool game that only uses one key made for [onekey](onekey.hackclub.com)

## features
- one key controls everything
- custom hero and assets instead of the normal bird
- same addictive flappy bird gameplay just better

## how to run it
just click the itch.io link above and play in browser, no install needed
